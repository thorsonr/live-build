# LiVE Pro Website v2

This folder contains an upgraded static website build for LiVE Pro.

- `index.html`: New information architecture and page structure.
- `styles.css`: Professional visual system and responsive layout.
- `app.js`: Functional job tracker, pipeline metrics, priority contacts, and local persistence.
